# NeLuaJ+

## Product
NeLuaJ+ is an Android Lua IDE / runtime based on AndroLua and LuaJ. Users write and run Lua scripts on device, with Material 3 UI, editor tooling, modules (file/http/saf/res), and embedded HTML docs.

## Users
Lua / Android hobbyists and plugin authors who need a local IDE with project management, layout helper, and API docs.
