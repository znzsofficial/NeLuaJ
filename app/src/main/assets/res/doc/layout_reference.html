布局表属性参考

属性与单位速查。用法、图片 src、样式构造见 loadlayout 专文；颜色优先 "?attr/…"。

相关文档 loadlayout 颜色与主题 MD3 组件 res 模块

1. 基本结构

return {
    ViewClass,              -- 第一个元素：View 类
    属性名 = 属性值,          -- 视图属性
    { ChildViewClass, ... } -- 子视图
}

2. 通用属性

尺寸

属性说明示例
layout_width宽度"match", "wrap", "200dp", "50%"
layout_height高度"match", "wrap", "100dp"
minWidth最小宽度"48dp"
minHeight最小高度"48dp"

尺寸值简写：

值等价于
"match" / "fill" / "-1"match_parent
"wrap" / "-2"wrap_content

尺寸单位：

后缀说明示例
dp密度无关像素"16dp"
sp缩放像素（文字专用）"14sp"
px像素"1px"
% / %w屏幕宽度百分比"50%"
%h屏幕高度百分比"30%h"
pt点"12pt"
in英寸"0.5in"
mm毫米"10mm"

间距

属性说明
padding四边内边距（可与单边属性组合；单边优先）
paddingLeft / paddingRight物理左/右内边距 → setPadding
paddingStart / paddingEndRTL 感知起始/末端 → setPaddingRelative
paddingTop / paddingBottom上/下内边距
layout_margin四边外边距（写入 leftMargin 等字段）
layout_marginLeft / Right / Top / Bottom单边；仅设置出现的边，不清零其它边
layout_marginHorizontal / layout_marginVertical左右 / 上下成对
layout_marginStart / layout_marginEndRTL → setMarginStart / setMarginEnd
layout_width / layout_heightmatch/wrap/dp → int 像素
layout_weightLinearLayout 权重（float）

Start / End（RTL）：

  • 单边 padding* / layout_margin* 只改写到的边，未出现的边保持原值。
  • 出现 paddingStart/paddingEnd 时用 setPaddingRelative；否则 setPadding。缺省边回退：单边 → padding 四边 → 当前 View padding。
  • layout_marginStart/End 走 setMarginStart/End；物理边写入 leftMargin 等字段。
  • layout_alignWithParentIfMissing 设置 RelativeLayout 的 alignWithParent 字段（不是 addRule）。
{
  MaterialTextView,
  text = "标题",
  paddingStart = "16dp",
  paddingEnd = "16dp",
  paddingTop = "12dp",
  paddingBottom = "12dp",
  layout_marginStart = "8dp",
  layout_marginEnd = "8dp",
}

可见性

属性值说明
visibility"visible"可见（默认）
"invisible"不可见但占位
"gone"不可见且不占位

visibility 为专用键：字符串走枚举表，也接受 0/4/8。

文本约束与输入

属性说明示例
singleLine单行（TextView）true
maxLines最大行数3
maxLength最大字符数（LengthFilter）20
inputType输入类型枚举，可 |"textMultiLine", "numberDecimal"
imeOptionsIME 选项"actionSearch", "actionDone"

高度与卡片（Material）

属性说明示例
elevation任意 View 的 Z 高度，支持 dp"4dp"
cardElevation / CardElevationMaterialCardView 卡片高度；非卡片时退化为 elevation0（MD3 推荐）
radius / cornerRadiusMaterialCardView.setRadius，支持 dp"16dp"
Checked / checked任意 android.widget.Checkable（Switch、CheckBox、Chip、RadioButton、MaterialCardView…）true
checkable / Checkable有 setCheckable(boolean) 的控件（Chip、MaterialCardView、MaterialButton…）；反射结果按 Class 缓存true
{
  MaterialCardView,
  layout_width = "match",
  layout_height = "wrap",
  radius = "16dp",
  cardElevation = 0,
  strokeWidth = "1dp",
  checkable = true,
  Checked = false,
  {
    MaterialTextView,
    text = "标题",
    maxLines = 2,
    padding = "16dp",
  },
}

背景

-- 颜色值
background = 0xFFFF0000          -- 整数颜色
background = "#FF0000"           -- 十六进制字符串

-- Drawable 对象

ID

4. 图片属性

属性说明示例
src图片源：Drawable / 路径 / URL / @drawableres.drawable("save", color)
scaleType缩放类型"centerCrop"
iconTint / TintColor图标着色（与 src 分开写）ColorUtil.getColorPrimary()
{
  AppCompatImageView,
  layout_width = "40dp",
  layout_height = "40dp",
  src = res.drawable("save", ColorUtil.getColorPrimary()),
}

@drawable/name 本身不着色；工具栏图标优先 res.drawable(name, color)。找不到资源会报错。

相关文档 loadlayout · src 详解 res.drawable

scaleType：

值说明

"centerCrop"等比缩放裁剪填满
"centerInside"等比缩放不超出
"fitXY"拉伸填满
"center"原始大小居中
"fitStart"等比缩放靠左上
"fitEnd"等比缩放靠右下

5. View 通用属性

以下属性可用于任何 View，直接通过 Java 反射设置：

属性说明示例
alpha透明度 (0.0~1.0)0.5
rotation旋转角度45
rotationX / rotationYX/Y 轴旋转30
scaleX / scaleYX/Y 轴缩放1.5
translationX / translationYX/Y 轴平移"10dp"
elevation高度（阴影）"4dp"
enabled是否启用true / false
clickable是否可点击true
focusable是否可获取焦点true
focusableInTouchMode触摸模式下可获取焦点true
selected是否选中true
clipToPadding是否裁剪到 paddingfalse
clipChildren是否裁剪子视图false
layerType图层类型"none", "software", "hardware"
layoutDirection布局方向"ltr", "rtl", "locale"
drawingCacheQuality绘制缓存质量"auto", "low", "high"
scrollbarStyle滚动条样式"insideOverlay", "outsideOverlay"
horizontalScrollBarEnabled水平滚动条true / false
verticalScrollBarEnabled垂直滚动条true / false
BackgroundResource背景资源 IDrippleRes
BackgroundColor背景颜色0xFFFF0000

属性与 Java API：常见 setXxx 可用同名属性，如 alpha = 0.5、enabled = false。BackgroundColor 对应 setBackgroundColor。

写错的属性名：对不上 setter / 字段时会在错误日志里提示。

枚举：如 inputType = "textMultiLine"、gravity = "center|end"，多个 flag 用 |。

颜色：支持 int、"#AARRGGBB"、"?attr/colorPrimary"。tint 类属性（iconTint 等）传入 int 会自动包成 ColorStateList。

8. 事件监听

所有以 on 开头的属性会被设置为事件监听器：

{
    MaterialButton,
    text = "点击",
    onClick = function(view)
        print("被点击了")
    end,
    onLongClick = function(view)
        print("被长按了")
        return true  -- 返回 true 表示已消费
    end,
}

常用事件：

属性说明
onClick点击
onLongClick长按
onTouch触摸
onFocusChange焦点变化
onItemClick列表项点击
onItemLongClick列表项长按
onCheckedChange选中状态变化
onScrollChange滚动变化

10. 列表与容器属性

ListView / Spinner

属性说明示例
items字符串数组，自动创建 Adapter{ "选项1", "选项2" }
DividerHeight分割线高度0, 1
FastScrollEnabled快速滚动true
{
    ListView,
    id = "list",
    items = { "苹果", "香蕉", "橙子" },
    DividerHeight = 0,
    onItemClick = function(parent, view, pos, id)
        print("点击了第", pos, "项")
    end,
}

ViewPager

属性说明示例
pagesView 列表{ view1, view2, view3 }
pagesWithTitle带标题的页面{ {view1, view2}, {"标题1", "标题2"} }
{
    ViewPager,
    id = "pager",
    pages = { page1View, page2View },
}

-- 带标题（配合 TabLayout）
{
    ViewPager,
    id = "pager",
    pagesWithTitle = {
        { page1View, page2View },
        { "首页", "设置" },
    },
}

ScrollView

属性说明示例
fillViewport内容不足时填满true
scrollbars滚动条"vertical", "horizontal", "none"

注意：ScrollView 只能有一个子视图，通常是 LinearLayout。

RecyclerView

RecyclerView 需要在 Lua 代码中设置 LayoutManager 和 Adapter：

import "androidx.recyclerview.widget.LinearLayoutManager"

recyclerView.setLayoutManager(LinearLayoutManager(activity))
recyclerView.setAdapter(adapter)

11. style 属性

通过 style 属性应用 Android 主题样式：

-- 使用主题属性引用
{ MaterialButton, text = "轮廓按钮",
  style = "?attr/materialButtonOutlinedStyle" }

-- 使用资源 ID
{ MaterialButton, text = "文字按钮",
  style = "?attr/borderlessButtonStyle" }
