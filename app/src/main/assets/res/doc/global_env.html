全局环境

每个 LuaActivity 脚本都运行在统一的全局环境中。下列名称可直接使用，无需额外 require。

对象

变量说明
this / activity当前 LuaActivity 实例
androidJavaPackage("android")，用于访问 Android 包
okHttp异步 OkHttp 封装对象
http同步网络模块
saf存储访问框架（SAF）模块

类

变量说明
Http异步网络相关 Java 类入口
R应用资源 ID 类

模块表

变量说明
okhttp同步网络模块表
jsonJSON 编解码
file文件读写与目录操作
res字符串、布局、字体等资源
ext二进制 pack / unpack

函数

函数说明
print输出到日志 / Toast 体系
printf格式化输出
loadlayout加载布局表
task异步任务
xTask基于协程的异步任务
thread后台线程
timer定时器
call主线程调用函数
lazy懒加载包装
dump序列化 Lua 表

示例

print(activity.getPackageName())
print(this.getLuaDir())

local body = okhttp.get("https://example.com")
local t = json.decode('{"a":1}')
file.save(this.getLuaPath("out.txt"), "hello")
