LuaFragmentAdapter

LuaFragmentAdapter 继承 FragmentStateAdapter，用于 ViewPager2。底层基于 RecyclerView，支持垂直滑动、RTL、更高效的页面回收。

基本用法

local Fragments = { fragment1, fragment2, fragment3 }

local adapter = LuaFragmentAdapter(activity, LuaFragmentAdapter.Creator {
  createFragment = function(i)
    return Fragments[i + 1] -- Java 索引从 0 开始
  end,
  getItemCount = function()
    return #Fragments
  end,
})

viewPager2.setAdapter(adapter)

刷新方法

方法说明
reload()刷新全部
reloadItem(position)刷新单个位置
changed(position, count)一段内容变化
inserted(position, count)插入一段
removed(position, count)移除一段
moved(from, to)移动页面

增删示例

table.insert(Fragments, newFragment)
adapter.inserted(#Fragments - 1, 1)

table.remove(Fragments, index)
adapter.removed(index - 1, 1)

adapter.reload()

position 使用 ViewPager2 索引，从 0 开始。

错误处理

createFragment / getItemCount 抛错时，会通过 activity.sendError("FragmentAdapter", e) 上报，并尽量回落到安全默认值，避免 ViewPager2 直接崩溃。
