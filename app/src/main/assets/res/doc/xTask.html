xTask

xTask 是 NeLuaJ+ 基于 Kotlin 协程封装的异步任务 API，适合在后台做 IO / 计算，再回到主线程更新 UI。

表参数写法（推荐）

xTask {
  -- dispatcher = "io", -- 可选：使用 IO 调度器
  task = function(coroutineContext)
    -- 后台执行
    local data = okhttp.get("https://example.com")
    return data
  end,
  callback = function(result)
    -- 主线程回调；成功时为返回值，失败时可能是错误信息
    print(result)
  end,
}

位置参数写法

xTask(
  function(coroutineContext)
    return "ok"
  end,
  function(result)
    print(result)
  end
  -- , "io"  -- 可选 dispatcher
)

参数说明

字段 / 参数说明
task后台执行的函数，可返回结果
callback可选；任务结束后的回调
dispatcher可选；传入 "io" 时使用 IO 调度器

注意

  • UI 操作应放在 callback 中，不要在 task 里直接改 View。
  • 可以省略 callback，仅在后台执行副作用逻辑。
  • 需要取消/更细粒度控制时，可结合返回的 job 封装（若当前版本暴露）自行管理。

与 task / thread 的区别

  • task：传统异步任务封装。
  • thread：直接开线程。
  • xTask：协程调度，写法更接近结构化异步。
