FastScrollerBuilder

基于 AndroidFastScroll，为 RecyclerView 添加快速滚动条。若适配器实现了 getPopupText（如 PopupRecyclerAdapter），可显示气泡提示。

依赖与前置

  • 目标为 RecyclerView。
  • 需要字母/标题气泡时，使用支持 getPopupText 的适配器。

示例

local FastScrollerBuilder = luajava.bindClass "me.zhanghai.android.fastscroll.FastScrollerBuilder"

FastScrollerBuilder(recyclerView)
  .useMd2Style()
  .setPadding(0, this.dpToPx(8), this.dpToPx(2), this.dpToPx(8))
  .build()

与 Popup 适配器配合

local adapter = PopupRecyclerAdapter(activity, PopupRecyclerAdapter.PopupCreator {
  getItemCount = function()
    return #dataList
  end,
  getPopupText = function(view, position)
    local name = dataList[position + 1].name or ""
    return utf8.sub(name, 1, 1)
  end,
  onCreateViewHolder = function(parent, viewType)
    local views = {}
    local holder = LuaCustRecyclerHolder(loadlayout(item_layout, views))
    holder.views = views
    return holder
  end,
  onBindViewHolder = function(holder, position, payloads)
    holder.views.title.text = dataList[position + 1].name
  end,
})
recyclerView.setAdapter(adapter)
FastScrollerBuilder(recyclerView).useMd2Style().build()

提示

  • useMd2Style() 使用圆角 MD2 风格滑块。
  • setPadding 可避免与列表内容或导航栏重叠。
  • 列表数据更新后无需重建 FastScroller，但适配器需正确通知数据集变化。
