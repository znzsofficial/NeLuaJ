LuaThemeUtil 字段参考

实现：github.daisukiKaffuChino.utils.LuaThemeUtil。实例：this.themeUtil。

本文只列字段。

用法与选型 颜色与主题 MD3 设计 
activity.dynamicColor()  -- 先启用色板
local t = this.themeUtil
local primary = t.primary.main

2. secondary / tertiary / error

4. background / outline / text

  • background.main / .on

5. 兼容 getter

getColorPrimary() ≡ primary.main；亦支持 colorPrimary 简写。任意 attr：

t.getAnyColor(MDC_R.attr.colorTertiaryContainer)

结构化字段（如 primary.main）与 getColorXxx() 指向同一套色值；两种写法都可以，新示例多采用结构化字段。
