FileObserver

使用 Android FileObserver 监听目录变化。NeLuaJ+ 可通过 override 覆盖 onEvent 实现。

示例

local FileObserver = luajava.bindClass "android.os.FileObserver"

local MyObserver = FileObserver.override {
  onEvent = function(superCall, event, path)
    switch event
     case FileObserver.CREATE
      print("created", path)
     case FileObserver.DELETE
      print("deleted", path)
     case FileObserver.MODIFY
      print("modified", path)
     default
      print("event", event, path)
    end
  end
}

local observer = MyObserver("/sdcard/Android")
observer.startWatching()

function onDestroy()
  observer.stopWatching()
end

要点

  • 必须在合适时机 startWatching()，并在 onDestroy 中 stopWatching()，避免泄漏。
  • path 参数通常是相对被监听目录的名称，不一定是绝对路径。
  • 事件在系统线程回调，更新 UI 时请切回主线程（如 call / Activity 方法）。
  • Android 10+ 对共享存储监听可能受限，优先监听应用可访问目录。

常见事件

常量含义
CREATE创建
DELETE删除
MODIFY修改
MOVED_FROM / MOVED_TO移动
