file 模块

file 提供常用文件与目录操作。路径支持绝对路径，也支持相对于工程 / 工作目录的路径（与当前运行上下文有关）。

API

file.readall(path)

local text = file.readall("main.lua")
local text2 = file.readall("/sdcard/test.lua")

读取整个文件为字符串。

file.save(path, content)

file.save("out.txt", "print('hello')")

写入文件；必要时创建父目录（视实现而定）。

file.exists(path)

if file.exists("init.lua") then
  print("exists")
end

file.type(path)

print(file.type("/sdcard")) -- "dir" 或 "file"

file.info(path)

local info = file.info("main.lua")
-- 常见字段：name、path、size、isDirectory 等（以实际返回表为准）

file.list(path)

local names = file.list("/sdcard")
for i, name in ipairs(names) do
  print(i, name)
end

列出目录下名称；内部通过表转换实现，大目录可能偏慢。

file.mkdir(path)

file.mkdir("/sdcard/NeLuaJ/demo")

创建目录及其缺失的父目录。

完整示例

local path = this.getLuaPath("cache/note.txt")
file.mkdir(this.getLuaPath("cache"))
file.save(path, "hello file module")
if file.exists(path) then
  print(file.readall(path))
  print(file.type(path), file.info(path))
end
